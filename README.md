# ⏱️ Stopwatch App

## 📌 Description
This is a simple Stopwatch Android application that measures elapsed time between start and stop actions. It works like a real handheld stopwatch.

## 🚀 Features
- Start, Stop, Reset functionality
- Real-time time tracking
- Displays time in MM:SS format
- Simple and user-friendly interface

## 🛠️ Technologies Used
- Kotlin
- Jetpack Compose
- Android Studio

## 📂 Project Structure
- MainActivity.kt – Contains UI and stopwatch logic

## ▶️ How to Run
1. Open project in Android Studio
2. Connect emulator or Android device
3. Click Run ▶️

## 🎯 Outcome
Developed a functional stopwatch app demonstrating basic state management and UI design.

## 📸 Output
Displays timer with Start, Stop, Reset buttons.
